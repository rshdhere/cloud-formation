## AWS CodeBuild VPC CloudFormation Template

This AWS documentation page provides a sample CloudFormation template for setting up a VPC for AWS CodeBuild. It covers a VPC layout with public and private subnets across two Availability Zones, internet access through an internet gateway, NAT gateways for private subnets, route tables, and a no-ingress security group.

Link:
https://docs.aws.amazon.com/codebuild/latest/userguide/cloudformation-vpc-template.html
